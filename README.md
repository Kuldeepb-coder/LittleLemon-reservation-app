# Little Lemon – Django Restaurant App 🍋

**Little Lemon** is a web application built with **Django**, simulating a restaurant management system.  
This project was developed as part of a capstone project, demonstrating backend web development, database handling, and basic user interaction with a restaurant reservation system.

---

## 🌟 Features

- **Table Reservation System:** Users can book tables for specific dates and times.  
- **Modular Django App Structure:** Organized with separate apps for scalability (`reservation` app included).  
- **Database Integration:** Uses Django ORM to manage reservations and potential future menu data.  
- **Local Development Ready:** Run the app locally with minimal setup.  
- **Version Controlled:** Fully tracked using Git and hosted on GitHub.

---

## 🛠 Technology Stack

- **Backend:** Python 3.x, Django 4.x  
- **Frontend:** HTML, CSS, Django Templates  
- **Database:** SQLite (default Django database)  
- **Development Tools:** VS Code, Git, GitHub, Virtual Environment (`venv`)
